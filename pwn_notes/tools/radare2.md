# radare2

## ¿Qué es?



## Instalación

Seguir los pasos de la [página oficial](https://rada.re/r/down.html).

## Uso

Para iniciar el programa debemos llamarlo así:

```bash
r2 <binary>
```

Por ejemplo para ver las direcciones de las funciones:

```radare2
aa
```
```radare2
afl
```
